## 🚀 快速开始

### 环境配置
1. **前端环境**: Node.js 16+, npm/yarn
2. **后端环境**: Go 1.21+, MySQL 8.0+
3. **开发工具**: Git, IDE (推荐VS Code)

### 启动步骤
```bash
# 1. 克隆项目
git clone <repository-url>
cd 云梦高校学生科研与竞赛项目管理系统的设计与实现

# 2. 启动后端
cd go-backend
go mod tidy
go run main.go

# 3. 启动前端
cd yunmeng-frontend
npm install
npm run dev
```
